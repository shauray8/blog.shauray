ORDERED SETS.

Defination: Let S be a set. An order on S is a relation, denotd by <, with the following properties.

1. if x ∈ S and y ∈ S then one and only one of the statements x < y, x = y, y < x is true.
2. if x,y,z ∈ S, if x < y and y < z, then x < z. its called the law of trichotomy

I feel like this is straight forward defination and it'll get clear with the examples ahead but to explain it in simple words an ordered set is a set S in which an order is defined.

Defination: S is an ordered set and E ⊂ S. if there exist a B ∈ S such that x ≤ B for every x ∈ E, we say that E is bounded above and B is called an upper bound of E.

this tells that for a set E ∃ B ∈ S such that B is greater than every element of E. ie. {1,2,3,4,5,6,7,8,9} this perticular set has 9 or 11 or 100 or 1000 as an upper bound.
and the same goes with the lower bound which is defined in the same way with ≥ in place of ≤. ie. the set we defined above has 1 or 0 or any -ve number as its lower bound.

Now with this concept we can further proof the existance of a Lowerst Upper Bound also called as supremum or sup and a Greatest Lower Bound also called as infimum or inf.
